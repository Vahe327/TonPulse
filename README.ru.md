## Что такое TonPulse?

TonPulse — это **полноценный DeFi-терминал** внутри Telegram. Профессиональные торговые инструменты, ИИ-аналитика и управление ликвидностью — всё в одном приложении на базе протокола **STON.fi** в блокчейне **TON**.

Каждый свап, каждая котировка, каждый маршрут проходит через STON.fi. TonPulse — грантовый проект **STON.fi Grant Program**.

## Возможности

### Торговля токенами
- **Обмен** любых TON-токенов через STON.fi с котировками в реальном времени
- **Мульти-хоп маршрутизация** — поиск лучшего пути через пулы ликвидности
- **Контроль проскальзывания** — настраиваемый допуск (0.5%, 1%, 3%, вручную)
- **Отображение ликвидности пула** — TVL и объём за 24ч перед обменом
- **Предупреждения о влиянии на цену** — цветовая индикация для крупных сделок
- **Экран подтверждения** — все детали перед подписанием транзакции

### Рыночные данные в реальном времени
- **Цены в реальном времени** через WebSocket — обновление каждые 15 секунд
- **Свечные графики** (1Ч / 4Ч / 1Д) на базе TradingView Lightweight Charts
- **OHLCV-данные** из GeckoTerminal с фоллбэком на пулы STON.fi
- **Анимация изменения цен** — зелёная/красная вспышка при обновлении
- **TVL и объём** отображаются для каждого токена
- **Трендовые токены** — отсортированы по ликвидности и активности

### Управление портфелем
- **Балансы TON + Жетонов** — автоматическое определение всех активов
- **Оценка в USD** — стоимость портфеля в долларах в реальном времени
- **Разбивка по токенам** — баланс, цена, стоимость, доля в портфеле
- **P&L за 24ч** — динамика портфеля за день
- **Кастомные жетоны** — иконки загружаются из метаданных блокчейна (IPFS поддерживается)

### Пулы ликвидности
- **Обзор пулов** — все пулы STON.fi с поиском и сортировкой
- **APR** — годовая доходность за 24ч, 7д, 30д
- **Добавление ликвидности** — односторонний или сбалансированный ввод
- **Вывод ликвидности** — снятие LP-позиций
- **Создание новых пулов** — объединение любых двух токенов
- **Отслеживание LP-позиций** — доля в пуле, заработанные комиссии, текущая стоимость
- **Предупреждения о непостоянных потерях**

### Покупка TON картой (Fiat On-Ramp)
- **Три провайдера** — Mercuryo (Visa/Mastercard), Neocrypto, Telegram Wallet
- **Открывается внешне** — безопасные страницы покупки через `openLink` / `openTelegramLink` Telegram
- **Пошаговые гайды** — раскрывающиеся инструкции для каждого провайдера (EN/RU)
- **Автоподстановка кошелька** — подключённый TON-адрес передаётся в URL провайдера
- **Триггер от ИИ** — ассистент определяет пустой кошелёк и предлагает купить TON

### Умный онбординг (От нуля до DeFi)

TonPulse проводит новичков от нулевого опыта до заработка на ликвидности:

1. **Создание кошелька** — пошаговая инструкция по настройке Tonkeeper прямо в чате
2. **Покупка TON** — оплата картой через Mercuryo, Neocrypto или Telegram Wallet
3. **Первый обмен** — ИИ-сопровождение с пояснениями
4. **Первый LP** — ИИ рекомендует лучший пул, объясняет риски, добавление в одно нажатие
5. **Установка алертов** — защита позиций уведомлениями о цене

ИИ определяет уровень пользователя (новичок/средний/продвинутый) и адаптирует рекомендации. 10 интерактивных карточек позволяют торговать прямо из чата.

### ИИ-ассистент (Groq LLM)
TonPulse использует три ИИ-модели для разных задач:

| Функция | Модель | Что делает |
|---------|--------|-----------|
| **Анализ токена** | llama-3.3-70b | Глубокий анализ с оценкой риска (1-10), сильные/слабые стороны, прогноз |
| **Бейджи риска** | llama-3.1-8b | Быстрая оценка риска 1-10 рядом с каждым токеном |
| **Инсайты свапа** | llama-3.1-8b | Одна строка ИИ-совета перед обменом (позитивный/нейтральный/негативный) |
| **Чат-ассистент** | llama-3.3-70b | Разговорный DeFi-советник с контекстом портфеля |

ИИ-ассистент **учитывает контекст** — он знает баланс кошелька, LP-позиции и текущую рыночную ситуацию. Даёт практические рекомендации с 10 интерактивными карточками:

| Карточка | Действие |
|----------|----------|
| **Купить TON** | Открытие fiat on-ramp с предзаполненным кошельком |
| **Обмен** | Выполнение свапа прямо из чата |
| **Добавить ликвидность** | LP-депозит в рекомендованный пул в одно нажатие |
| **Установить алерт** | Создание ценового алерта из диалога |
| **Инфо о токене** | Детальная карточка анализа токена |
| **Портфель** | Сводка портфеля с позициями |
| **Рекомендация пула** | ИИ-подобранные лучшие пулы с APR |
| **Обучение** | DeFi-концепции простым языком |
| **Подключить кошелёк** | Предложение подключить кошелёк |
| **Подтверждение** | Подтверждение транзакции перед подписанием |

**Лимит:** 10 ИИ-запросов в час на пользователя. Результаты кешируются в Redis.

### Ценовые алерты
- **Целевые цены** — уведомление когда токен пересекает заданный порог (выше/ниже)
- **Уведомления в Telegram** — алерты приходят прямо в чат
- **Повторяющиеся алерты** — срабатывают каждый раз, не только один
- **Управление** — создание, просмотр, удаление на вкладке Алерты

### Telegram-нативный интерфейс
- **Тактильная отдача** — вибрация на каждое взаимодействие
- **Тёмная тема** — премиальный тёмный интерфейс, совпадает с Telegram
- **Safe area** — корректная работа с клавиатурой и viewport
- **Плавные анимации** — spring-переходы через Framer Motion
- **Pull-to-refresh** — нативный жест обновления
- **Нижняя навигация** — 5 вкладок (Рынок, Обмен, Портфель, ИИ, Настройки)

### Мультиязычность
- **Английский** и **Русский** — полный перевод интерфейса через i18next
- ИИ-ответы также на языке пользователя
- Переключатель языка в Настройках

## API-эндпоинты

### Токены
```
GET  /api/v1/tokens                          — Список токенов с ценами
GET  /api/v1/tokens/{address}                — Детали токена + статистика
GET  /api/v1/tokens/{address}/ohlcv          — Данные для свечного графика
```

### Обмен
```
GET  /api/v1/swap/pairs?token={address}      — Торговые пары
GET  /api/v1/swap/quote                      — Котировка обмена
POST /api/v1/swap/build-transaction          — Сборка транзакции обмена
```

### Портфель
```
GET  /api/v1/portfolio?wallet={address}      — Полный портфель с позициями
```

### Ликвидность
```
GET  /api/v1/pools                           — Обзор пулов (поиск, сортировка, пагинация)
GET  /api/v1/pools/{address}                 — Детали пула
POST /api/v1/liquidity/simulate              — Симуляция добавления ликвидности
POST /api/v1/liquidity/build-transaction     — Сборка LP-транзакции
GET  /api/v1/liquidity/positions             — LP-позиции пользователя
```

### ИИ
```
POST /api/v1/ai/analyze-token               — Глубокий анализ токена
POST /api/v1/ai/risk-scores                  — Пакетная оценка рисков
GET  /api/v1/ai/swap-insight                 — Настроение по свапу
POST /api/v1/ai/chat                         — Чат с ИИ-ассистентом
```

### Алерты и пользователь
```
GET    /api/v1/alerts                        — Список алертов
POST   /api/v1/alerts                        — Создать алерт
DELETE /api/v1/alerts/{id}                   — Удалить алерт
GET    /api/v1/user                          — Профиль
PATCH  /api/v1/user                          — Обновить настройки
```

### Реальное время
```
WS   /api/v1/ws/prices                      — Стрим цен в реальном времени
```

## Быстрый старт (Разработка)

### Требования
- Docker и Docker Compose
- Node.js 20+
- Python 3.11+

### 1. Клонирование и настройка

```bash
git clone https://github.com/your-org/tonpulse.git
cd tonpulse
cp .env.example .env
# Отредактируйте .env — укажите ваши API-ключи
```

### 2. Запуск через Docker Compose

```bash
docker compose up -d
```

Запускает: PostgreSQL, Redis, Backend (FastAPI), Frontend (Vite), Nginx.

### 3. Миграции БД

```bash
docker compose exec backend alembic upgrade head
```

### 4. Открытие приложения

- Разработка: `http://localhost:5173`
- Через Telegram: укажите URL Mini App вашего бота

## Продакшен-деплой

### Вариант А: PM2 (VPS / bare metal)

#### Бэкенд

```bash
cd backend
pip install -r requirements.txt
alembic upgrade head
pm2 start "uvicorn app.main:app --host 0.0.0.0 --port 8000" --name tonpulse-backend
```

#### Фронтенд

```bash
cd frontend
npm ci
npm run build
pm2 start serve-prod.cjs --name tonpulse-frontend
```

Production-сервер (`serve-prod.cjs`) раздаёт собранную папку `dist/` на порту 5173:
- SPA-фоллбэк (все роуты отдают `index.html`)
- Долгосрочное кеширование хешированных ассетов (JS, CSS, шрифты)
- Минимальное потребление памяти

#### Nginx

Настройте Nginx для проксирования:
- `/api/*` и `/ws/*` → `http://127.0.0.1:8000`
- Всё остальное → `http://127.0.0.1:5173`

```bash
# SSL через Let's Encrypt
sudo certbot --nginx -d ваш-домен.com
```

### Вариант Б: Docker Compose

```bash
cp .env.example .env
# Укажите продакшен-значения
docker compose up -d --build
docker compose exec backend alembic upgrade head
```

### Настройка Telegram-бота

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Включите Mini App: Bot Settings → Menu Button → Set URL
3. Укажите URL `https://ваш-домен.com`

## Соответствие гранту STON.fi

TonPulse создан для **STON.fi Grant Program ($10,000)**. Все требования выполнены:

| Требование | Реализация |
|-------------|---------------|
| Интеграция STON.fi SDK | Все свапы через `@ston-fi/sdk` и HTTP API STON.fi |
| DeFi-продукт | Полный торговый терминал: обмен, портфель, ликвидность, алерты |
| Чёткие майлстоуны | Модульный код, каждая функция тестируется независимо |
| Долгосрочное видение | Расширяемая архитектура, ИИ-функции, мультиязычность |

**Ценность для STON.fi:** Каждый пользователь TonPulse генерирует торговый объём на STON.fi. ИИ-ассистент активно рекомендует пулы и свапы через протокол.
